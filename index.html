java集合类（四）Map集合和Collections类相关操作 

    Map用于保存具有映射关系的数据，map中有两组值，一组保存key，key不允许重复；一组保存value，value可以重复。
    Map集合与Set集合非常相似，事实上，java实现实现一个Map，再通过包装一个所有value都为null的Map实现的Set，因此Set下的常用实现类，Map
基本都有对应，如HashMap，TreeMap,SortedMap等。
1.方法
    clear（）：清空所有键值对。
    containsKey(Value):Map是否包含Key(Value).
    get(key):返回key对应的value。
    put(key,value):添加一个键值对。
    remove（key）：删除key对应的键值对。
    getKey（Value）：返回包含的key，value。

示例代码：

    import java.util.HashMap;
    import java.util.Map;

    public class MapTest {
        public static void main(String[] args){
            Map map = new HashMap();
            map.put("疯狂java讲义",109);
            map.put("疯狂ios讲义",10);
            map.put("疯狂讲义",79);
            System.out.println( map.put("疯狂ios讲义",79));
            System.out.println(map);
            System.out.println("是否包含iOS"+map.containsKey("疯狂ios讲义"));
            System.out.println("是否录入价格"+map.containsValue(99));
            System.out.println("iOS的价格"+map.get("疯狂ios讲义"));
            map.remove("疯狂讲义");
            System.out.println(map);

        }
    }

2.HashMap，HashTable类
    与HashSet相似，这两类判断key相等也是要hashcode值相等和equals（）返回true。

示例代码：

    import javax.management.ObjectName;
    import java.util.Hashtable;

    class A{
        int count;
        public A(int count){
            this.count=count;
        }
        public boolean equals(Object obj){
            if(this==obj){
                return true;
            }
            if(obj!=null && obj.getClass()==A.class){
                A a=(A)obj;
                return this.count==a.count;
            }
            return false;
        }

        @Override
        public int hashCode() {
            return this.count;
        }
    }
    class B{
        @Override
        public boolean equals(Object obj) {
            return true;
        }

4.TreeMap实现类
    与TreeSet相似，也有自然排序和定制排序，提供方法顺序访问元素。

    示例代码：

    import java.util.TreeMap;

    class R implements Comparable{
        int count;
        public R(int count){
            this.count=count;
        }
        public String toString(){
            return "R[count:"+count+"]";//toString函数使在输出时，输出想要的名称
        }

        @Override
        public boolean equals(Object obj) {
            if(this==obj)
                return true;
            if(obj!=null&&obj.getClass()==R.class){
                R r=(R)obj;
                return r.count==this.count;
            }
            return false;
        }

        @Override
        public int compareTo(Object o) {
            R r=(R)o;
            return count>r.count?1:count<r.count?-1:0;
        }
    }
    public class TreeMapTest {
        public static void main(String[] args){
            TreeMap tm=new TreeMap();
            tm.put(new R(3),"ASD");
            tm.put(new R(5),"ASDfg");
            tm.put(new R(4),"ASDf");
            System.out.println(tm);
            System.out.println(tm.firstEntry());
            System.out.println(tm.lastKey());
            System.out.println(tm.higherKey(new R(2)));
            System.out.println(tm.subMap(new R(-1),new R(4)));//返回范围内的子Map.
        }
    }

java集合类（三）List集合和Queue集合相关操作 

List集合
    代表一个元素有序，可重复的集合，集合中每个元素都有对应的顺序索引，因此允许出现重复元素，ArrayList是其典型实现。
    1.方法
        void add(int index,Object e):添加元素到index处。
        Object get(int index):返回index处的元素。
        int indexOf(Object e):位置索引。
        Object remove（int index）:删除并返回index处的元素。
        Object set(int index,Object e):替换，返回被替换对象。

示例代码：
    import java.util.ArrayList;
    import java.util.List;

    public class ListTest {
        public static void main(String[] args){
            List books = new ArrayList();
            books.add(new String("轻量级java ee企业实战"));
            books.add(new String("疯狂java讲义"));
            books.add(new String("疯狂安卓讲义"));
            System.out.println(books);
            books.add(1,new String("ajax讲义"));
            for(int i=0;i<books.size();i++){
                System.out.println(books.get(i));
            }
            books.remove(2);
            System.out.println(books);
            books.set(2,"疯狂java讲义");
            System.out.println(books);
        }
    }

--------------------------------------------------------------------------------------------------------

    List判断对象相等只要通过equals()方法进行比较即可。
    java为List增加了sort()方法和replaceAll()方法，sort的参数是Comparator对象，可以用lambda表达式，而replaceAll()参数是UnaryOperator对象,而
UnaryOperator同样为函数式接口，所以也可以用lambda表达式。

示例代码：
    import java.util.ArrayList;
    import java.util.List;

    class Atest{
        public boolean equals(Object obj){
            return true;
        }
    }
    public class ListTest2 {
        public static void main(String[] args){
            List books = new ArrayList();
            books.add(new String("轻量级java ee企业实战"));
            books.add(new String("疯狂java讲义"));
            books.add(new String("疯狂安卓讲义"));
            books.add(1,new String("ajax讲义"));
            books.sort((o1,o2)->((String)o1).length()-((String)o2).length());//代码块中只有一条语句，不用return关键词
            System.out.println(books);
            books.replaceAll(ele->((String)ele).length());
            System.out.println(books);
            books.remove(new Atest());//list判断两对象相等只比较equals函数，
            System.out.println(books);//调用atest类的equals函数总是返回true,删除时list将元素依次进行比较.

        }
    }

Queue类
    用于模拟队列这种数据结构，遵循先进先出的规则。
    1.方法
    add（Object e）：将指定元素加入队列尾部。
    element（）：获得头部元素。
    peek():获取头部元素。
    pool（）：获取头部元素并删除。
    2.标准实现类-PriorityQueue
        该类为较标准的队列实现类，但是保存顺序不是按加入队列的顺序，而是按队列元素大小排序，peek（）或pool（）取出的是最小的元素。

    示例代码：
        import java.util.PriorityQueue;

        public class PriorityQueueTest {
            public static void main(String[] args){
                PriorityQueue pq =new PriorityQueue();
                pq.offer(-6);
                pq.offer(-3);
                pq.offer(7);
                pq.offer(3);
                System.out.println(pq);
                System.out.println(pq.poll());
            }
        }

java集合类（二）Set相关操作 

Set集合与Collection基本相同，只不过Set不允许出现重复元素。

1.HashSet类
HashSet类是Set接口的典型实现，在使用他进行数据储存时，会通过调用该对象的hashCode（）方法，equals（）方法来判断是否与集合中元素重复。

示例代码：
    import java.util.HashSet;
    import java.util.Iterator;

    class R{
        int count;
        public R(int count){
            this.count=count;
        }
        public String toString(){
            return "r[count:"+count+"]";    //改写toString方法，是对象按规定格式输出。
        }
        public boolean equals(Object obj){
            if(this==obj){        //对象相同的情况
                return true;
            }
            if(obj!=null&&obj.getClass()==R.class){ //非空且类型一致
                R r=(R)obj;
                return this.count==r.count;
            }
            return false;

        }
        public int hashCode(){
            return this.count; //将count作为hashcode检测值，正常情况下有标准的方法
        }
    }
    public class HarshsetTest {
        public static void main(String []args){
            HashSet hs=new HashSet();
            hs.add(new R(5));
            hs.add(new R(-3));
            hs.add(new R(9));
            hs.add(new R(-2));
            System.out.println(hs);
            Iterator it=hs.iterator();
        }
    }

--------------------------------------------------------------------------------------------------------

2.LinkedHashSet类
他是HashSet类的子类，在其基础上使用链表维护元素的次序，会按照元素添加的顺序来访问集合中的元素。

示例代码：
    import java.util.LinkedHashSet;

    public class LinkedHashSetTest {
        public static void main(String[] args){
            LinkedHashSet books =new LinkedHashSet();
            books.add("dsd");
            books.add("sdsf");
            System.out.println(books);
            books.remove("dsd");
            books.add("dsd");
            System.out.println(books);//元素顺序总是于添加顺序相同
        }
    }

3.TreeSet类
    该类是SortedSet接口的实现类，保证集合元素处于排序状态。    
    在原有基础上提供了如下方法：
        first():返回集合第一个函数。
        last():返回集合最后一个函数。
        higher(Object e)：返回大于指定元素的最小元素.
        lower(Object e):返回小于指定元素的最大元素。
        headSet(Object e):返回集合中小于e的子集.
        tailSet(Object e):返回集合中大于等于e的子集.

示例代码：
    import java.util.TreeSet;

    public class TreeSetTest {
        public static void main(String[] args){
            TreeSet nums= new TreeSet();//根据数值大小排序
            nums.add(5);
            nums.add(2);
            nums.add(10);
            nums.add(-9);
            System.out.println(nums);
            System.out.println(nums.first());
            System.out.println(nums.last());
            System.out.println(nums.headSet(4));//排在四前面的
            System.out.println(nums.tailSet(5));//排在五后面的
            System.out.println(nums.subSet(-3,4));//-3和4之间的
        }
    }

同时，TreeSet支持两种排序方式：自然排序和定制排序。

（1）自然排序
    调用Comparable接口的compareTo(Object e)方法，比较大小关系，对集合元素升序排序，同时判断元素相等唯一标准
也是compareTo方法是否返回0（但也要保证equals（）返回结果一致）。
    因此，在将对象存入TreeSet之前，该对象的类必须先实现Comparable接口，显然，同类才能比较大小。

示例代码：
    import java.util.TreeSet;

    class Z implements Comparable{
        int age;
        public Z(int age){
            this.age=age;
        }
        public boolean equals(Object obj){
            return true;
        }

        @Override
        public int compareTo(Object o) {
            return 1;
        }
    }
    public class TreeSetTest2 {
        public static void main(String[] args){
            TreeSet set= new TreeSet();
            Z z1=new Z(6);
            set.add(z1);
            System.out.println(set.add(z1));
            System.out.println(set);
            ((Z)(set.first())).age=9;
            System.out.println(((Z)(set.last())).age);
        }
    }

（2）定制排序
    定制排序通过使用Comparator接口中的int compare（o1,o2）方法，该方法返回正整数，代表o1>o2,返回0，代表相等，返回负数，代表o1小于o2。
    Comparator为函数式接口，可以直接使用lambda表达式。

示例代码：
    import java.util.TreeSet;

    class M{
        int age;
        public M(int age){
            this.age=age;
        }
        public String toString(){
            return "M[age:"+age+"]";
        }
    }
    public class TreeSetTest3 {
        public static void main(String[] args){
            TreeSet ts =new TreeSet((o1,o2)->{
                M m1=(M)o1;
                M m2=(M)o2;
                return m1.age>m2.age?-1:m1.age<m2.age?1:0;
            });
            ts.add(new M(5));
            ts.add(new M(-3));
            ts.add(new M(9));
            System.out.println(ts);
        }
    }

java集合类（一）Collection相关操作 

Collection是java集合框架的根接口，同时也是set,queue和list的父接口。
1.方法
    boolean add(object obj):添加一个元素。
    void clear（）：清空集合。
    boolean contains(object obj):是否包含指定元素。
    boolean isEmpty():集合是否为空。
    boolean remove(object obj):移出指定元素。

下列程序将展示如何通过上述方法操作集合元素。
    public class CollectionTest {
        public static void main(String [] args){
            Collection c = new ArrayList();
            c.add("孙悟空");
            c.add(6);
            System.out.println("number of item:"+c.size());
            c.remove(6);
            System.out.println("number of item:"+c.size());
            System.out.println("是否包含孙悟空字符:"+c.contains("孙悟空"));
            Collection books = new HashSet();
            books.add("dasfasa");
            books.add("dssdfsf");
            System.out.println("c集合是否全包含books："+c.containsAll(books));
            c.removeAll(books);
            c.clear();
        }
    }

--------------------------------------------------------------------------------------------------------

2.遍历集合的两种方法
    （1）Lambda表达式遍历集合元素
        调用forEach（Consumer action）方法,其中Consumer是函数式接口，可以试用lambda表达式来遍历。
    （2）使用Iterator遍历集合元素
        Iterator是专门用来遍历Collection中元素，也叫迭代器，并没有盛放对象的能力，在进行迭代时，并不是把集合元素传入，而是
    把集合元素的值传入，所以修改迭代变量并不会对集合元素产生影响。

下列程序将展示如何通过上述方法遍历集合。    
    public class lambda遍历集合 {
        public static void main(String[] args){
            Collection books = new HashSet();
            books.add("as");
            books.add("sd");
            books.add("df");

            books.forEach(obj->System.out.println("迭代元素："+obj));

            Iterator it = books.iterator();//iterator迭代器
            while(it.hasNext()){
                String book =(String)it.next();
                System.out.println(book);
                if(book.equals("as")){
                    it.remove();
                }
                it.forEachRemaining(obj->System.out.println("元素"+obj));
            }
            System.out.println(books);
        }
    }

3.对集合的Stream操作
    通过对stream流式api的操作，可以简单的完成一些复杂的功能。
    步骤为：使用builder（）建立对应的流 -> 调用add()添加元素 -> 调用build（）获得对应的Stream -> 调用对应的聚集方法

    示例代码：
        import java.util.stream.IntStream;

        public class IntStreamTest {
            public static void  main(String[] args){
                IntStream is = IntStream.builder()
                        .add(20)
                        .add(13)
                        .add(-2)
                        .add(18)
                        .build();
                IntStream newIs =is.map(ele->ele*ele+2);//中间方法使用后返回流
                System.out.println(is.max().getAsInt());//在对流使用末端方法后，该流被消耗。
                //System.out.println(is.min().getAsInt());
                //System.out.println(is.sum());
                //System.out.println(is.count());
                //System.out.println(is.average());
                //System.out.println("is中所有元素的平方和都大于20："+is.allMatch(ele->ele*ele>20));

            }
        }

    Stream提供了大量的方法用于聚集操作，这些方法分为中间方法（返回值为流，可以继续调用后续方法）和末端方法（使用后流被消耗）。
    常见的中间方法有：
        （1）filter（Predicate predicate）:过滤所有不符合predicate条件的元素。
        （2）sorted():用于排序。
    末端方法有：
        (1)forEach(Consumer action）:遍历所有元素，对所有元素执行action。
        (2)min(),max()
        (3)count():返回流中元素数量

    而与此同时，有专门的的stream（）方法，可以返回该集合对应的流，大大丰富了集合的功能.
    示例代码：
        import java.util.Collection;
        import java.util.HashSet;

        public class CollectionStream {
            public static void main(String[] args){
                Collection books = new HashSet();
                books.add("dasfasa");
                books.add("dssdfsfsfadwd");
                System.out.println(books.stream()
                .filter(ele->((String)ele).contains("das")).count()
                );
                System.out.println(books.stream()
                .filter(ele->((String)ele).length()>10).count());
                books.stream().mapToInt(ele->((String)ele).length()).forEach(System.out::println);
            }
        }
