java集合类（四）Map集合和Collections类相关操作 

    Map用于保存具有映射关系的数据，map中有两组值，一组保存key，key不允许重复；一组保存value，value可以重复。
    Map集合与Set集合非常相似，事实上，java实现实现一个Map，再通过包装一个所有value都为null的Map实现的Set，因此Set下的常用实现类，Map
基本都有对应，如HashMap，TreeMap,SortedMap等。
1.方法
    clear（）：清空所有键值对。
    containsKey(Value):Map是否包含Key(Value).
    get(key):返回key对应的value。
    put(key,value):添加一个键值对。
    remove（key）：删除key对应的键值对。
    getKey（Value）：返回包含的key，value。

示例代码：

    import java.util.HashMap;
    import java.util.Map;

    public class MapTest {
        public static void main(String[] args){
            Map map = new HashMap();
            map.put("疯狂java讲义",109);
            map.put("疯狂ios讲义",10);
            map.put("疯狂讲义",79);
            System.out.println( map.put("疯狂ios讲义",79));
            System.out.println(map);
            System.out.println("是否包含iOS"+map.containsKey("疯狂ios讲义"));
            System.out.println("是否录入价格"+map.containsValue(99));
            System.out.println("iOS的价格"+map.get("疯狂ios讲义"));
            map.remove("疯狂讲义");
            System.out.println(map);

        }
    }

2.HashMap，HashTable类
    与HashSet相似，这两类判断key相等也是要hashcode值相等和equals（）返回true。

示例代码：

    import javax.management.ObjectName;
    import java.util.Hashtable;

    class A{
        int count;
        public A(int count){
            this.count=count;
        }
        public boolean equals(Object obj){
            if(this==obj){
                return true;
            }
            if(obj!=null && obj.getClass()==A.class){
                A a=(A)obj;
                return this.count==a.count;
            }
            return false;
        }

        @Override
        public int hashCode() {
            return this.count;
        }
    }
    class B{
        @Override
        public boolean equals(Object obj) {
            return true;
        }
    }
    public class HashTableTest {
        public static void main(String[] args){
            Hashtable ht =new Hashtable();
            ht.put(new A(6000),"asd");
            ht.put(new A(1234),"qwer");
            ht.put(new A(46675),new B());
            System.out.println(ht);
            System.out.println(ht.containsValue("cghhnnn"));
            System.out.println(ht.containsKey(new A(6000)));
            ht.remove(new A(46675));
            System.out.println(ht);
        }
    }

3.LinkedHashMap实现类
    使用双向链表维护元素插入顺序，保证迭代顺序与插入顺序一致。

示例代码：

    import java.util.LinkedHashMap;

    public class LinkedHashMapTest {
        public static void main(String[] args){
            LinkedHashMap score=new LinkedHashMap();
            score.put("a",23);
            score.put("b",24);
            score.put("c",25);
            score.forEach((key,value)->System.out.println(key+"->"+value));

    }

4.TreeMap实现类
    与TreeSet相似，也有自然排序和定制排序，提供方法顺序访问元素。

    示例代码：

    import java.util.TreeMap;

    class R implements Comparable{
        int count;
        public R(int count){
            this.count=count;
        }
        public String toString(){
            return "R[count:"+count+"]";//toString函数使在输出时，输出想要的名称
        }

        @Override
        public boolean equals(Object obj) {
            if(this==obj)
                return true;
            if(obj!=null&&obj.getClass()==R.class){
                R r=(R)obj;
                return r.count==this.count;
            }
            return false;
        }

        @Override
        public int compareTo(Object o) {
            R r=(R)o;
            return count>r.count?1:count<r.count?-1:0;
        }
    }
    public class TreeMapTest {
        public static void main(String[] args){
            TreeMap tm=new TreeMap();
            tm.put(new R(3),"ASD");
            tm.put(new R(5),"ASDfg");
            tm.put(new R(4),"ASDf");
            System.out.println(tm);
            System.out.println(tm.firstEntry());
            System.out.println(tm.lastKey());
            System.out.println(tm.higherKey(new R(2)));
            System.out.println(tm.subMap(new R(-1),new R(4)));//返回范围内的子Map.
        }
    }

    5.Collections工具类
    该类提供了大量方法对集合元素进行操作。

示例代码:
    import java.util.ArrayList;
    import java.util.Collection;
    import java.util.Collections;

    public class SortTest {
        public static void main(String[] args){
            ArrayList nums=new ArrayList();
            nums.add(2);
            nums.add(-5);
            nums.add(3);
            nums.add(1);
            System.out.println(nums);
            Collections.reverse(nums);//翻转
            System.out.println(nums);
            Collections.sort(nums);//排序
            System.out.println(nums);
            Collections.shuffle(nums);//随机排序
            System.out.println(nums);
        }
    }

    同时还可以创建不可变集合，当然也可以使用java9新增的.of()方法来创建不可变集合

示例代码：

    import java.util.*;

    public class UnmodifiableTest {
        public static void main(String[] args){
            List a = Collections.emptyList();//不可变
            Set b = Collections.singleton("asd");//不可变
            Map scores =new HashMap();
            scores.put("qw",80);
            scores.put("er",82);
            Map c = Collections.unmodifiableMap(scores);//此时不能够做出任何改变了
            Set set =Set.of("qw","sa","sd");
            System.out.println(set);
            List list = List.of(12,34,56,78);//使用of可以快速的创建多个元素的set，list，map不可变集
            System.out.println(list);
            Map map=Map.of("zx",23,"as",34);
        }
    }
